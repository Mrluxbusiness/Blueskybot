## ✅ নতুন Strategy: "80/20 Value-First Marketing"

> **মূলনীতি:** 80% পোস্ট = ফ্রি Value দাও (tips, hacks, motivation)
> 20% পোস্ট = সরাসরি Course promote করো

### কেন এটা কাজ করবে?
1. **Trust Build হয়** — মানুষ দেখে তুমি genuinely জানো
2. **Algorithm Boost** — Value posts বেশি engage হয়, reach বাড়ে
3. **Warm Audience তৈরি হয়** — যখন sell করো, তারা already তোমাকে চেনে
4. **Bluesky Culture** — Bluesky হচ্ছে anti-spam platform, value-first approach দরকার

## ⏰ Posting Schedule (Optimized)

### কেন ৫ বারের বদলে ৩ বার?
- Bluesky তে over-posting = spam flag risk
- Quality > Quantity
- প্রতিটা পোস্ট আলাদা account থেকে হলেও, ৩ বার optimal

### প্রস্তাবিত Schedule:

```
Slot 1: 08:00 AM EST  → ☀️ Morning (Value Post — Category A)
Slot 2: 01:00 PM EST  → 🍔 Afternoon (Curiosity/Soft Sell — Category B/C)  
Slot 3: 07:00 PM EST  → 🌆 Evening (Direct Offer/Engagement — Category D/E)
```

### Weekly Content Calendar:

| দিন | Slot 1 (Morning) | Slot 2 (Afternoon) | Slot 3 (Evening) |

কোডে `link facet` নেই — তাই URL clickable হয় না।

### সমাধান:
1. **Bluesky Link Facet যোগ করতে হবে** — URL auto-detect করে clickable বানাবে
2. **Course Links:**
   - TikTok Course: `[আপনার Gumroad/Landing Page URL]`
   - Instagram Course: `[আপনার Gumroad/Landing Page URL]`
3. **Link Placement Rules:**
   - Category A (Value): ❌ কোনো লিংক নেই
   - Category B (Curiosity): ⚠️ মাঝে মাঝে "link in my profile" hint
   - Category C (Social Proof): ✅ শেষে subtle link
   - Category D (Direct): ✅ Bold link with CTA
   - Category E (Engagement): ❌ কোনো লিংক নেই

## 🔄 Content Rotation System

### কেন দরকার?
- একই পোস্ট বারবার দিলে spam — Bluesky ban করতে পারে
- DeepSeek দিয়ে প্রতিবার unique text generate হবে
- কিন্তু angle/structure fixed থাকবে

### কিভাবে কাজ করবে:

```
1. GitHub Actions trigger হবে (scheduled time)
2. Script দেখবে আজ কোন দিন (Mon/Tue/...)
3. সেই দিনের জন্য নির্ধারিত Category + Angle select করবে
4. Time slot অনুযায়ী সঠিক angle pick করবে
5. DeepSeek কে detailed prompt দিয়ে unique text generate করাবে
6. Course-specific image generate হবে
7. ৫টি account-এ একসাথে post হবে
8. কিন্তু প্রতিটি account ভিন্ন variation পাবে (same angle, different text)
```
